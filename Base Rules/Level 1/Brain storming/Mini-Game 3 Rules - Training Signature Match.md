# Mini-Game 3 Rules - Training Signature Match

> الهدف من الملف: تحديد قواعد واضحة للميني جيم الثالثة بحيث تكون مبنية على نتائج Mini-Game 1 و Mini-Game 2، مع مبرر قصصي جديد بدون تأخير رسالة الجد.

---

## 1) Core Rules

- لازم يبقي فيه مبرر قصصي 
- الميني جيم الثالثة هي **اختبار تحقق** وليست تدريب جديد بالكامل.
- رسالة الجد تظهر كاملة بعد Mini-Game 2 بشكل طبيعي.
- التحديات تتولد من نفس أنماط الأخطاء السابقة (dynamic based on history).
- أي تحديث لقيم الروبوت يتم مرة واحدة في النهاية فقط.

---

## مقترح

### Rule A - Movement Echo

- النظام يعيد سيناريوهات مشابهة لأخطاء Mini-Game 1 (drift/camera correction).
- المطلوب: تقليل الخطأ المتكرر وسرعة تصحيح أعلى من قبل.

### Rule B - Energy/Path Echo

- النظام يقدم مسارات متعددة مشابهة لـ Mini-Game 2.
- المطلوب: اختيار منطقي بين (أقصر/أأمن/أوفر طاقة) حسب الحالة.

### Rule C - Auto-Pilot Check

- الروبوت يتحرك ذاتيًا لفترة قصيرة (30-45 ثانية).
- اللاعب لديه عدد تدخلات محدود (افتراضي 3 Overrides).
- كل Override إضافي يقلل التقييم.
